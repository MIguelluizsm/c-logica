# Sistema de Média de Notas em C

Este programa calcula a média de três notas e informa se o aluno foi:
- Aprovado
- Recuperação
- Reprovado

O usuário pode repetir o cálculo quantas vezes quiser.

## Conceitos praticados
- Entrada de dados
- Condicionais (if / else)
- Estrutura de repetição (do while)
- Saída formatada

## Execução
gcc media.c -o media
./media
